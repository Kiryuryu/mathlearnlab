## 使用说明

每条错题包含以下字段。坚持记录，复习时事半功倍。

### 字段说明

| 字段 | 说明 |
|------|------|
| **编号** | 统一编号，如 GSKJ-001 (高数-极限-001) |
| **日期** | 做题日期 |
| **来源** | 真题年份+题号 / 模拟卷名称 / 习题集页码 |
| **题型** | 选择题 / 填空题 / 解答题 |
| **知识点** | 核心考察概念 |
| **我的答案** | 写下当时错误的解法和思路 |
| **正确答案** | 标准解答过程 |
| **错误类型** | 计算失误 / 概念混淆 / 思路错误 / 审题不清 / 公式记错 |
| **教训** | 一句话总结，下次如何避免 |
