## 级数破局心法：从死记判别到理解"速度"

### 核心思维：收敛就是"项趋于 0 得够快"

每一项都趋近于 0 是必要条件但不充分。调和级数 Σ1/n 的项趋近于 0 但发散——因为趋近得**不够快**。

### 三步框架

#### 第一步：看通项，猜收敛性

拿到级数 Σa_n，先看：
1. a_n → 0 吗？如果不，立刻发散（必要条件）
2. a_n 趋近于 0 的速度像 1/n^p 吗？
   - p>1 → 很可能收敛
   - p≤1 → 很可能发散

#### 第二步：选武器

| 判别法 | 适合什么样的级数 |
|--------|------------------|
| 比值判别 | 含阶乘 n!、指数 a^n 的级数 |
| 根值判别 | 含 (·)^n 结构，通项有 n 次方 |
| 比较判别 | 能直接和已知敛散性的级数比大小 |
| 极限比较 | 通项和 1/n^p 同阶时，极限法确认 |
| 积分判别 | 单调递减的正项级数（对应反常积分） |
| 交错级数判别 | 正负交替且单调递减→收敛 |

> **心法**：比值判别是"放大镜"——前后项的比值告诉你项缩小的速度。比值<1 → 按比例缩小→收敛；比值>1 → 越来越大→发散。

#### 第三步：幂级数——找收敛"地盘"

幂级数 Σa_n(x-c)^n 在"收敛半径"内收敛。

求收敛半径 R：
- 比值法：R = lim|a_n/a_{n+1}|
- 根值法：R = 1/lim|a_n|^(1/n)

在|x-c|<R 内绝对收敛，在|x-c|>R 外发散。半径端点处需要单独判断。

### 泰勒展开：函数的多项式面孔

把函数写成泰勒级数：

e^x = 1 + x + x²/2! + x³/3! + ...
sinx = x - x³/3! + x⁵/5! - ...
cosx = 1 - x²/2! + x⁴/4! - ...
ln(1+x) = x - x²/2 + x³/3 - ...
(1+x)^α = 1 + αx + α(α-1)x²/2! + ...

> 记忆心法：e^x 全加，sinx 奇次交错，cosx 偶次交错，ln 没有阶乘。

### 傅里叶级数：周期 = 正弦波叠加

a_0/2 + Σ(a_n·cos(nx) + b_n·sin(nx))

计算系数：a_n = (1/π)∫f(x)cos(nx)dx，b_n = (1/π)∫f(x)sin(nx)dx

奇延拓 → 正弦级数（只有 sin 项），偶延拓 → 余弦级数（只有 cos 项）。

### 万能口诀

> 先看通项趋不趋零，再比 p 级数断死活；
> 阶乘指数比值法，n 次方幂根值判；
> 幂级半径端点判，泰勒展开到同阶。
