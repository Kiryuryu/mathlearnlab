## 导数破局心法：从公式记忆到几何直觉

### 核心思维：导数就是变化的快慢

f'(x) 回答三个问题：
1. 函数在 x 处是上升还是下降？（符号）
2. 上升/下降有多快？（大小）
3. 变化率本身在加速还是减速？（二阶导）

### 三步框架

#### 第一步：认清求导对象

| 结构 | 策略 |
|------|------|
| 多项式/幂函数 | 直接降幂：d/dx x^n = n x^(n-1) |
| 乘积 f(x)·g(x) | 乘积法则：(fg)' = f'g + fg' |
| 商 f(x)/g(x) | 商的法则：(f/g)' = (f'g - fg')/g² |
| 复合 f(g(x)) | 链式法则：f'(g(x))·g'(x)，从外到内逐层剥 |
| 隐函数 | 两边同时求导，解出 dy/dx |
| 参数方程 | dy/dx = (dy/dt)/(dx/dt) |

#### 第二步：理解导数的"语言"

**切线的斜率**：f'(a) 是在 x=a 处曲线的最贴近直线的斜率。线性近似 f(x)≈f(a)+f'(a)(x-a) 是微积分最基本的工具。

**相关变化率**：一个量变化时，另一个量如何跟着变？都是链式法则的应用。

#### 第三步：中值定理家族

| 定理 | 含义 |
|------|------|
| Rolle | 两端相等，中间必有导数为零的点 |
| Lagrange | 存在某点导数等于平均变化率 |
| Cauchy | 两个函数的变化率在某处成比例 |

> **心法**：中值定理的精髓是"存在性"——它告诉你一定存在某个点满足某条件，但不告诉你是哪个点。题目问"证明存在某点使..."，立刻想中值定理。

### 单调性与极值

f'(x)>0 → 上升，f'(x)<0 → 下降。

**找极值的三步**：
1. 求导，令 f'(x)=0 找驻点
2. 用二阶导判断：f''>0 极小，f''<0 极大，f''=0 进一步判断
3. 检查边界和不可导点

### 凸性与拐点

f''>0 凹向上（碗形），f''<0 凹向下（拱形）。f'' 变号的点 = 拐点。

### 万能口诀

> 一阶看升降，二阶判凹凸；
> 链式剥洋葱，乘积各导和；
> 极值找驻点，拐点看变号。
