## 积分破局心法：从背公式到看结构

### 核心思维：积分就是"反着求导"

每个积分公式都对应一个求导公式的逆向。理解"为什么这样积"比记住"怎么积"重要一百倍。

### 三步框架

#### 第一步：观察结构，选择武器

拿到积分先扫一眼结构：

| 结构特征 | 方法 |
|----------|------|
| 明显是某函数的导数 | 直接写出原函数（如∫e^x dx = e^x） |
| 多项式×三角函数 | 分部积分，多项式选为 u（反对幂指三） |
| 复合函数 f(g(x))·g'(x) | 凑微分，令 u=g(x) |
| 根号√(a²-x²) | 三角代换 x=a·sinθ |
| 根号√(x²+a²) | 三角代换 x=a·tanθ |
| 根号√(x²-a²) | 三角代换 x=a·secθ |
| 有理分式 P(x)/Q(x) | 部分分式分解 |
| 三角函数的复杂组合 | 万能代换 t=tan(x/2)（万不得已） |

#### 第二步：分部积分法

公式：∫u dv = uv - ∫v du

**"反对幂指三"口诀**：选择 u 的优先级——反三角 > 对数 > 幂函数 > 指数 > 三角。靠前的选为 u（因为求导后会变简单）。

**经典配对**：
- ∫x·e^x dx → u=x, dv=e^x dx
- ∫x·sinx dx → u=x, dv=sinx dx  
- ∫lnx dx → u=lnx, dv=dx
- ∫e^x·sinx dx → 分部两次后回到原积分，解方程

#### 第三步：定积分的武器

**对称性**：奇函数在 [-a,a] 上的积分为 0，偶函数可以折半。

**换元必换限**：做变量替换时，积分上下限也要一起变。

**Wallis 公式**：∫[0,π/2] sin^n x dx = ∫[0,π/2] cos^n x dx，n 奇偶公式不同，直接套。

### 反常积分：有限还是无限？

判断 ∫[a,∞) f(x)dx 收敛还是发散，核心是比较。

**p-积分**：∫[1,∞) 1/x^p dx → p>1 收敛，p≤1 发散。这是比较的基准。

**比较判别法**：拿一个已知收敛/发散的积分作对比。

### 万能口诀

> 一看复合凑微分，二遇乘积分部积；
> 根号里面平方和，三角代换最给力；
> 定积对称用奇偶，反常拿 p 作基准。
