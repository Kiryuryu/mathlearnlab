# 错题本

> 记录、分析、反思。每道错题都是一次理解加深的机会。

---

## 使用说明

做练习时遇到不会的或做错的题目，记录下来。定期回看，你会惊讶于自己的进步。

| 字段 | 说明 |
|------|------|
| 日期 | 做题日期 |
| 主题 | 极限 / 导数 / 积分 / 级数 / 多元 |
| 题目 | 原题描述 |
| 我的思路 | 写下的解法和想法 |
| 正确思路 | 正确的解法 |
| 错误类型 | 计算失误 / 概念混淆 / 思路错误 |
| 学到了什么 | 一句话总结 |
